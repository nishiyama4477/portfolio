# Calculate distance between A and B by latitude and longitude

A地点の緯度・経度、B地点の緯度・経度から2点間の距離を求める

## 実装している主な機能

* 2地点の緯度と経度を入力することで2地点間の距離が分かる
## 環境

* python 3.8.5
